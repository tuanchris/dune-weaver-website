## Getting Help

### Support Channels
1. **Documentation:** Check docs first
2. **Discord:** Ask the community
3. **GitHub Issues:** Report bugs
4. **Email Support:** Last resort for complex issues

### Best Practices
When asking for help:
- Describe the issue clearly
- Include what you've tried
- Share relevant settings
- Provide photos/videos if helpful
- Specify your hardware version
