# Gallery Images

This folder contains community gallery images for each Dune Weaver table type.

## Folder Structure

```
public/gallery/
├── dwmp-mini-pro/    # DWMP Mini Pro images
├── dwp-pro/          # DWP Pro images
├── dw-mini/          # DW Mini images
└── og-dw/            # OG DW Original images
```

## How to Add Images

1. Choose the appropriate folder for your table type
2. Add your images with descriptive filenames (e.g., `spiral-pattern.jpg`, `geometric-design.png`)
3. Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`
4. Rebuild the site to see your new images

## Image Guidelines

- **Resolution**: Minimum 800x800px recommended
- **Aspect Ratio**: Square (1:1) works best
- **File Size**: Keep under 2MB for optimal loading
- **Naming**: Use descriptive names with hyphens (e.g., `fibonacci-spiral-2024.jpg`)

## Notes

- Images are automatically lazy-loaded for better performance
- Alt text is generated from the filename
- Images are displayed in alphabetical order by filename
